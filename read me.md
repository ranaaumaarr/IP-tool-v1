# IP Address Management Tool

A beginner-friendly Python networking utility that calculates basic IPv4 subnet information using CIDR notation. This project helps understand fundamental networking concepts such as IP addressing, subnet masks, network IDs, and broadcast addresses.

---

## Features (Version 1)

- Calculate Network Address
- Calculate Broadcast Address
- Display Subnet Mask
- Display Total Number of Addresses

---

## Technologies Used

- Python 3
- Built-in `ipaddress` module
- Visual Studio Code (Recommended)

---

## Requirements

### Software Requirements

- Python 3.8 or later
- Visual Studio Code (optional but recommended)

### Python Libraries

No external libraries are required.

This project uses Python’s built-in module:

```python
import ipaddress
```
